## Course Content

| Day | Topic | Description |
| :-: | ----- | ----------- |
| 1 | **S** — Single Responsibility | Each class should have only one reason to change |
| 2 | **O** — Open/Closed | Open for extension, closed for modification |
| 3 | **L** — Liskov Substitution | Subtypes must be substitutable for their base types |
| 4 | **I** — Interface Segregation | No class should be forced to implement methods it doesn't use |
| 5 | **D** — Dependency Inversion | Depend on abstractions, not on concrete implementations |
| 6 | Final Project | A comprehensive project combining all five principles |
